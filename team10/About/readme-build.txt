This project was developed in Python 3.7.2.
To build this project, first make sure python 3.7 is installed on your system.
Then install pipenv. The Instruction can be found here: (https://pipenv.readthedocs.io/en/latest/install/#installing-pipenv).
Or if you have python pip on your system you can run the following command:
py -3.7 -m pip install pipenv
After this, all the dependencies can easily be installed using pipenv. To do this run the following command in the "Build" directory:
py -3.7 -m pipenv install
The project is now built and can be run using the instruction found in the readme-users.txt
